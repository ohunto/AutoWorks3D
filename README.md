# AutoWorks3D

High-quality 3D-printed automotive solutions — tools, discontinued parts, and custom fixes.

**Live site:** [autoworks3d.com](https://autoworks3d.com)

Static site hosted on **GitHub Pages** with a custom domain registered through **Namecheap**. The `CNAME` file binds the repository to the domain; the matching DNS records live in the Namecheap dashboard.

## Editing

The entire page is `index.html` — a single-file static page with inline CSS. Push to the `main` branch and GitHub Pages redeploys automatically.
